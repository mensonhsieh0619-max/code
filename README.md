# FHIR 競賽專案

本專案依照競賽規則建立，包含規劃文件、實作內容及學習紀錄。

## 目錄結構說明

- `規劃文件/`: **規劃文件**
  - 包含情境敘述、目標、需求分析、工作流程、使用角色及主要 FHIR Resource。
- `實作內容/`: **FHIR 基礎實作**
  - `規格/`: 資料規格文件。
  - `程式原始碼/`: 程式碼（可由 AI 輔助產生）。
  - `執行結果/`: 執行結果與 HAPI FHIR Server 讀取/上傳證明。
- `學習紀錄/`: **學習紀錄文件**
  - `影片筆記/`: 教學影片觀看筆記。
  - `AI互動紀錄/`: 與 AI 互動之說明與截圖。
  - `導師會議紀錄/`: 與 weaver（導師）諮詢紀錄。
  - `reflection.md`: 學習心得（需親自撰寫，不可由 AI 代寫）。

## 繳交提醒
1. 實作資料須上傳至 GitHub 並設為公開。
2. FHIR 資料須確保可上傳至 HAPI FHIR Server。
3. 學習心得至少 500 字，且**禁止 AI 代寫**。
